# MyGlucos

This is a progressive web for monitoring glucose blood sugar for patients

## Requirements

1. Nodejs(+ npm)
2. stenciljs-CLI
3. firebase-tools


## Contributors

DeveloperPrince 

## Development Deployment

start development Server
```bash
npm start
```

## Build

```bash
npm run build
```

## Stage & Production on Firebase

initialize Firebase project

```bash
firebase init
```
Firebase Deploy

```bash
firebase deploy
```
